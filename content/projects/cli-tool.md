A command-line tool that accelerates project initialization with pre-configured setups. Features:

- Interactive prompts for project configuration
- Multiple template support (React, React + TanStack, Next.js)
- Automatic dependency installation
- Git repository initialization with conventional commit setup
- ESLint, Prettier, and Husky pre-configured
- Published on npm with automated releases
