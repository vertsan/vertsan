A clean, performant portfolio website built from the ground up using TanStack Start for SSR and TanStack Router for file-based routing. Features include:

- Dark/light theme with system preference detection and FOUC prevention
- Responsive design with mobile-first approach
- Content management via content-collections with Markdown frontmatter
- AI-powered resume assistant for interactive Q&A
- Optimized Core Web Vitals with SSR and route-based code splitting
