An accessible, themeable component library built on Radix UI primitives. Features:

- 30+ production-ready components with consistent API design
- Comprehensive Storybook documentation with interactive playrooms
- Full keyboard navigation and screen reader support (WCAG 2.1 AA)
- Customizable theming with CSS variables and dark mode support
- Automated visual regression testing with Chromatic
- Published as an npm package with tree-shaking support
